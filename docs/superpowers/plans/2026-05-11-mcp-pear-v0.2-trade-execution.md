# mcp-pear v0.2 — Trade Execution Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add 10 write MCP tools to mcp-pear (position open/close/adjust, leverage, risk params, cancel order, agent wallet) behind a hard `PEAR_TRADE_ENABLED=true` gate. Bump to v0.2.0.

**Architecture:** New `requireTradeEnabled()` on `PearConfig` short-circuits every write tool before any network call. PearClient gains 10 new methods, each a thin wrapper over Bearer-JWT POST/PUT/DELETE. Tools all follow the same shape: zod input → gate → call → render. No client-side signing (Pear signs server-side).

**Tech Stack:** TypeScript 5.6, Node ≥22, zod 3.x, FastMCP 3.x. No new runtime deps.

**Spec:** [`docs/superpowers/specs/2026-05-11-mcp-pear-v0.2-trade-execution.md`](../specs/2026-05-11-mcp-pear-v0.2-trade-execution.md)

```markdown
## v0.2 — trade execution (write tools)

Ten new tools that execute trades on your behalf. **All are off by default**; set `PEAR_TRADE_ENABLED=true` in your env to enable them.

| Tool | Type | Description |
|---|---|---|
| `get_agent_wallet`     | read  | Get the agent wallet Pear uses to sign your trades. |
| `create_agent_wallet`  | write | Create one. After creation, approve it on Hyperliquid. |
| `open_position`        | write | Open a pair position (MARKET / TRIGGER / TWAP / …). |
| `close_position`       | write | Close one position by id. |
| `close_all_positions`  | write | Close every open position. |
| `adjust_position`      | write | Reduce / increase position size 1–100 %. |
| `adjust_leverage`      | write | Set leverage 1–100x on an existing position. |
| `set_risk_parameters`  | write | Set or update TP / SL on a position. |
| `cancel_order`         | write | Cancel a pending limit / TP / SL order. |
| `cancel_twap_order`    | write | Cancel a TWAP order + remaining chunks. |

| Env var | Default | Description |
|---|---|---|
| `PEAR_TRADE_ENABLED` | `false` (unset) | Set to `"true"` to enable the ten write tools. Strictly `"true"` — any other value keeps them disabled. |
```
